## [1.0.0] - 2026-05-19

### Added
- ✨ Versão inicial do PowerBI IA
- 🐍 Script Python para descompactar PBIX
- 🐍 Script Python para recompactar PBIX
- 🐍 Script Python para validar integridade
- 📡 Node.js MCP Server para edição de layout
- 📚 Documentação completa (8 arquivos)
- 🎯 Exemplos práticos prontos para usar
- 🔧 Suporte a edição de colors, fontes, tamanhos
- 📍 Suporte a reorganização de layout

### Features
- Descompactar PBIX em JSONs editáveis
- Conectar Claude/Copilot via MCP
- Editar propriedades visuais em tempo real
- Recompactar PBIX com mudanças
- Validar integridade de arquivo

### Tech Stack
- Python 3.8+
- Node.js 16+
- MCP SDK
- Claude/GitHub Copilot
